# PWA Icons

The PWA requires icon files. Place these files in the public/ directory:

- pwa-192x192.png (192x192 pixels)
- pwa-512x512.png (512x512 pixels)

You can:
1. Use an online tool to convert SVG to PNG
2. Use a design tool like Figma, Canva, or Photoshop
3. Use ImageMagick: convert icon.svg -resize 192x192 pwa-192x192.png

For now, a simple SVG placeholder is included.
